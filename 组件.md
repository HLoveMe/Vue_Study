* data、computed、watch、methods

	* data必须是一个函数 data的值为函数返回值得一个副本

* 组件必须在全局进行注册'

* 方法传递 v-on == @

	* 组件模板中定义方法名称 "$event('event-name',param)"
	* 组件绑定函数

	```
	<template>
		<div :click="$event('countAdd')">
			AAAAA
		</div>
		<div :click="$event('reduce',100)">
			BBBBB
		</div>
	</template>
	{
		name:"SomeView"
	}
	<SomeView  @coungtAdd="aa"/ @reduce="bb($event)" />
	```
* v-model

	* v-model 相当于两条命令

		```
		<input  v-model='name'/>

		<input

		 	v-on:input="name = $event"
		 />
		```
